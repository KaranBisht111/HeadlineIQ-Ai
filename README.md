# 🧠 HeadlineIQ AI

HeadlineIQ AI is an intelligent NLP-powered headline generation platform that analyzes textual content and generates optimized headlines using Natural Language Processing techniques.

---

## 🚀 Features

- AI-powered headline generation
- Multiple headline styles
- Sentiment analysis
- Keyword extraction
- Named entity recognition
- SEO-oriented analytics
- Modern responsive UI
- Real-time text processing

---

## 🛠️ Tech Stack

### Backend
- Python
- Flask
- NLTK
- spaCy
- TextBlob
- Scikit-learn

### Frontend
- HTML5
- CSS3
- JavaScript

---

## 📊 NLP Features

- TF-IDF keyword extraction
- Sentiment detection
- Text normalization
- Named entity recognition
- Dynamic headline synthesis

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/KaranBisht111/headlineiq-ai.git
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Install spaCy Model

```bash
python -m spacy download en_core_web_sm
```

### Run Project

```bash
python app.py
```
